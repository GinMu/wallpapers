# wallpapers
a simple wallpapers-spider demo based on python3 and Scrapy-1.3

* 基于Scrapy-1.3图片下载python爬虫
* git clone https://github.com/GinMu/wallpapers.git
* brew install pyenv
* pyenv install 3.x
* pip install scrapy
* cd wallpapers
* pyenv local 3.x
* cd wallpapers
* scrapy runspider WallPapersSpider.py
